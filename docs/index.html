FanzyZones

Native window-zone snapping for macOS. Like FancyZones, in Swift.

Download DMG View on GitHub

Install with Homebrew

brew tap benwbooth/fanzyzones
brew trust benwbooth/fanzyzones
brew install --cask benwbooth/fanzyzones/fanzyzones
Drag to snapHold Shift and drag a window into a zone — or auto-snap on any drag.
Keyboard shortcuts⌃⌥←/→ to cycle zones, ⌃⌥1–9 to snap directly.
Visual menuClick a pane in a layout to snap the focused window there.
Custom layoutsDraw, resize, and split panes; save named layouts.
Per-displayAssign a different layout to each monitor.
ConfigurableModifier keys, gaps, overlay color, launch at login.

Permissions

FanzyZones needs Accessibility access to move windows. It prompts on first launch — enable it in System Settings → Privacy & Security → Accessibility. The app is signed but not notarized; on first open, right-click → Open.
